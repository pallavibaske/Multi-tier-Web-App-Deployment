# 🚀 Multi-tier Web App Deployment

## 🎯 Project Overview

This project demonstrates the deployment of a scalable three-tier web application on AWS.

The application is divided into three layers:

- Frontend (Presentation Layer)
- Backend (Application Layer)
- Database (Data Layer)

This architecture improves security, scalability, and maintainability.

---

## 🛠 AWS Services Used

- Amazon EC2
- Amazon RDS
- Amazon VPC
- Security Groups
- Internet Gateway
- Route Tables

---

## 🏗 Architecture

The frontend is hosted on an EC2 instance and communicates with the backend application. The backend securely connects to an Amazon RDS database inside the VPC.

---

## 🔄 Workflow

1. User accesses the frontend application.
2. Frontend sends requests to the backend.
3. Backend processes requests.
4. Backend communicates with Amazon RDS.
5. Database returns the requested data.

---

## ✨ Features

- Three-tier Architecture
- Secure Database Layer
- Scalable Design
- High Availability
